# Event App Tracker Monorepo

This is a production-ready monorepo for the Event App Tracker project.

## Structure

- `apps/` — Application entry points (frontend, backend)
- `packages/` — Shared code (types, utilities)
- `infrastructure/` — Deployment and platform configs
- `database/` — Migrations and DB scripts
- `docs/` — Documentation
- `scripts/` — Automation scripts

See `docs/` for more details.
